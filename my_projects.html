01 Python

For this first project, I managed to successfully process multiple csv files containing data about the weather, converting them into meaningful text-based summaries. To do so, I created several functions in order to: convert temperatures, format dates, calculate mean, max and min values, store data with specific parameters as well as create general and daily summaries formatted with specific instructions. I then run every function against set tests to ensure that code written had the expected behavior and could be correct for a variety of different inputs.
